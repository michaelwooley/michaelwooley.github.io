About This Site

This site was built with Hugo, a static website generator. It is hosted on GitHub.

The basic theme is based on tufte-css, a stylesheet based on the Edward Tufte’s books and handouts. The css was converted into a full-blown Hugo theme by the creators of hugo-tufte.

I have begun an overhaul of the hugo-tufte theme that includes:

  • Inline and marginal images and divs with captioning.
  • Sharing buttons (work in progress).
  • Easy addition of custom javascript and css for demos.
  • Various tweaks to the layouts and css.

Once I make some more progress I’ll look into merging these changes with hugo-tufte. In the meantime, I’d be happy to share over email.
